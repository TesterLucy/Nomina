# 🇨🇴 Sistema de Liquidación de Nómina Colombiana

### 🚀 Descripción General
Este software es una solución web integral diseñada para la gestión y cálculo automatizado de nómina bajo la normativa laboral vigente en Colombia (Actualizado 2025-2026). El sistema transforma el complejo proceso de liquidación laboral en una experiencia simple, precisa y visualmente intuitiva.

La aplicación procesa de forma exacta todos los componentes del salario, desde el devengado básico hasta las deducciones tributarias de ley, garantizando cumplimiento matemático y legal.

---

### ✨ Características Principales
*   **Cálculo de Devengado:** Procesa Salario Mínimo Mensual Legal Vigente (SMMLV), Auxilio de Transporte y horas extra.
*   **Deducciones de Ley:** Automatización de aportes a Salud y Pensión (4% cada uno) y cálculo preciso del IBC.
*   **Módulo de Retención en la Fuente:** Implementación avanzada del **Artículo 383 del Estatuto Tributario**, utilizando la tabla marginal expresada en **UVT** para una tributación exacta.
*   **Interfaz Modernizada:** Diseño basado en **Glassmorphism**, ofreciendo una estética profesional, limpia y "juvenil" que facilita la lectura de datos financieros.
*   **Totalmente Responsivo:** Optimizado para funcionar correctamente en dispositivos móviles, tablets y computadoras de escritorio.

---

### 🛠️ Detalles Técnicos
*   **Cálculo de UVT:** Valores parametrizados para el periodo fiscal actual.
*   **Seguridad:** Sistema de login integrado para protección de datos básicos.
*   **Tecnologías:** Desarrollado con estándares modernos de HTML5, CSS3 (Custom Properties y Flexbox/Grid) y JavaScript Vanilla.

---

### ⚖️ Normativa Aplicada
El sistema se rige por las leyes vigentes del Ministerio de Trabajo y la DIAN en Colombia, incluyendo topes salariales, bases de cotización y rangos de retención para ingresos laborales.
